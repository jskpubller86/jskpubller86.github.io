title : printf를 이용한 출력
categories : language c
---


#### 10자리에 맞춰서 숫자 출력, 앞 자리는 공백으로 채움.

```c
printf("%10d", 5);
//         5
```


#### 10자리에 맞춰서 숫자 출력, 앞 자리는 0으로 채움.

```c
printf("%010d", 5);
//0000000005
```


#### 10자리에 맞춰서 숫자 출력, 앞 자리는 0으로 채움.

```c
printf("%10h", 5);
//0000000005
```
