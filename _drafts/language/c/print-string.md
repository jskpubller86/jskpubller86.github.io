title : printf를 이용한 출력
categories : language c
---

C에서 기본적은 출력을 담당하는 함수로 printf가 있다.

printf의 포맷 옵션은 다음과 같다.

> printf("%[+/-][0]w[.n][l]변환문자", 인수);

참고로 괄호는 [] 옵션을 의미한다.

인수없이 출력할 수 있다.

<br>

#### 인수를 10진수로 출력

```c
printf("%d", 5);
//5

printf("%d", 5.1);
//5
```

<br>

#### 인수를 10자리에 맞춰서 10진수 출력, 앞 자리는 공백으로 채움.

```c
printf("%10d", 5);
//         5
```

<br>

#### 10자리에 맞춰서 숫자 출력, 앞 자리는 0으로 채움.

```c
printf("%010d", 5);
//0000000005
```
