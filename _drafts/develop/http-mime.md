# mimeType
마임타입(mimeType)은 클라이언트(client)에게 문서의 다양성을 알려주기 위한 메커니즘이다.

## 마임타입(mimeType)의 종류

| file         | mime               |
|--------------|--------------------|
| txt          | text/plain         |
| html, htm    | text/html          |
| csv          | text/csv           |
| jpeg, jpg    | image/jpeg         |
| png          | image/png          |
| mpeg         | audio/mpeg         |
| mp4          | vedio/mp4          |
